sbt-lighter

  • Github: pishen/sbt-lighter
  • A SBT plugin which lets you run your Spark jobs on AWS EMR easily.
  • This project is a successor to KKBOX/spark-deployer, which was widely adopted in KKBOX. Learned from the collected feedback, sbt-lighter has a simpler design and is much more easier to use.

scalikejdbc-generic

  • Github: pishen/scalikejdbc-generic
